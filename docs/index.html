Overview

システム概要

SharedShop Plusは、実務に近いWebアプリケーション開発を学ぶためのECサイトプラットフォームです。

🎯

開発目的

Java/Spring Bootを用いた標準的なWebアプリケーションの構築手法を学び、保守性の高いコードを書くスキルを習得します。

✨

特徴

直感的なUI、論理削除によるデータ保持、厳密な在庫管理など、実際のビジネスロジックを考慮した設計になっています。

💻

利用イメージ

PC快適に操作可能。管理者画面も充実しており、商品の管理から注文状況の把握まで一目瞭然です。

Core Features

主な機能

ECサイトとして必要な機能を網羅。セッション管理やバリデーションなど、細部まで丁寧に実装されています。

👤

会員登録・ログイン

アカウント作成からログイン、プロフィール管理まで。セキュリティを考慮した実装。

🔍

商品検索・詳細

商品名、カテゴリでの絞り込み。見やすい詳細ページで商品の魅力を伝えます。

🛒

買い物かご

セッションを利用したカート機能。在庫状況に合わせた動的な個数調整もサポート。

📦

注文・履歴

複数ステップの注文フロー。過去の注文内容もマイページからいつでも確認可能。

⚙️

商品・会員管理

管理者専用のダッシュボード。データの登録・編集・削除をGUIで簡単に行えます。

🛡️

権限・アクセス制御

ユーザー権限に応じたページアクセス制限。Filterを用いた堅牢な認可管理。
